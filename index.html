Game of Samsara

This adaptation of the Game of Life extends the game beyond death through reincarnation.
  • An atman's karma is determined by the number of dead cells and alive cells around it throughout its lifetimes.
  • A cell dies if it has reached the end of its lifespan.
  • An atman can only be rebirthed in a cell if the cell is empty.
  • An atman can only be rebirthed in a cell if it has at least two living neighbors.
  • An atman can only be rebirthed in a cell if it is in the same caste based on accumulated karma.
  • Unlike the Game of Life, the game is constantly running rather than evaluating the whole board at an interval.
