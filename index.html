Game of Samsara

This adaptation of the Game of Life extends the game beyond death through reincarnation.

A cell dies if it has <2 or >8 living neighbors. When it dies, its atman looks for a new cell to occupy.

An atman can be rebirthed in a cell if —

  • the cell is empty.
  • the cell has at least two living neighbors.
  • the cell is in the same caste (based on accumulated karma).

An atman's karma is determined by the number of dead cells and alive cells around it throughout its lifetimes.

Unlike the Game of Life, the game is constantly running rather than evaluating at discrete time steps.
